## Development

### Prerequisites

* Node v16.20
* npm v8.19

### Setup

1. Install the dependencies with `npm install`

2. Run the site with `npm start`

3. Ensure the API is listening on http://localhost:3000/ and browse to
   http://localhost:4200/tickets/
